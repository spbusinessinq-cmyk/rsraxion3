━━━━━━━━━━━━━━━━━━━━━━━

Replace vague scoring with a clear multi-factor model.

Use probabilistic / multi-factor / weighted scoring language.

Create or refine scoring functions:

A. Relevance Score 0–100

relevanceScore =
  domainWeight * 0.25 +
  institutionalImpact * 0.20 +
  systemImpact * 0.20 +
  crossDomainPotential * 0.20 +
  recencyWeight * 0.15

B. Confidence Score 0–100

confidenceScore =
  sourceReliability * 0.35 +
  corroborationStrength * 0.25 +
  domainConsistency * 0.15 +
  signalClarity * 0.15 +
  recencyIntegrity * 0.10

C. Threat Score 0–100

threatScore =
  severity * 0.30 +
  systemicImpact * 0.25 +
  geographicSpread * 0.15 +
  domainCoupling * 0.20 +
  escalationVelocity * 0.10

Map threat score:
0–30 LOW
31–49 GUARDED
50–69 ELEVATED
70–84 HIGH
85–100 CRITICAL

D. Priority Score 0–100

priorityScore =
  relevanceScore * 0.30 +
  confidenceScore * 0.20 +
  threatScore * 0.30 +
  recencyWeight * 0.10 +
  sourceReliability * 0.10

Use priorityScore to select primary signals.
